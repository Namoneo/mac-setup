
# Namoneo Mac Setup

One-command macOS developer environment setup.

## Includes

- Homebrew
- Node (nvm)
- pnpm / bun
- Angular CLI
- Python
- Docker
- Redis
- PostgreSQL
- VSCode
- Cursor
- AI CLI tools

## Install

Run:

curl -fsSL https://raw.githubusercontent.com/Namoneo/mac-setup/main/install.sh | bash

After ~10 minutes your Mac will be fully ready for development.
